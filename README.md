# Actividad 8 - Algoritmos Voraces (Greedy)

## Descripción

Implementación en Java de dos problemas utilizando la estrategia Greedy:

1. Mochila Fraccional
2. Cobertura de Antenas

## Requisitos

- Java JDK 8 o superior

## Compilación

```bash
javac MochilaFraccional.java
javac CoberturaAntenas.java
```

## Ejecución

```bash
java MochilaFraccional
java CoberturaAntenas
```
